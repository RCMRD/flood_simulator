The CREST stream flow viewer and flood simulator has the capability to generate a flood map from gauged values for basins that do not have CREST and from CREST outputs. To simulate flood using the gauged values for any basin, here are the steps to follow:

  1. On the left hand side menu, select the country
  2. Select the basin by name
  3. Select the simulation source to be gauge value
  4. Check the icon for manual input
  5. On the space provided, enter the gauge value( these values refers to the water levels recorded everyday by water agencies)
  6. Click on simulate floods
The last step will take a while, and once done an option of downloading the map will appear

To simulate floods using CREST values (these only apply for Nzoia and Malaba basins so far), follow the below steps:

  1. Step 1 and 2 above
  2. Select the simulation source as CREST values-
  3. A chart range time query option will appear- use it to query the time period from start date to end date
  4. Click on update graph- A graph showing the values of discharges and stage versus time will appear on the top section
  5. To simulate a flood map, click on the line representing the stage value for a specific date. The applications will automatic update the flood determination based on the threshold.
  6. If the stage surpasses the threshold, click on simulate flood
Then once done you can download the map The application also comes with an option to download time series data CREST output in csv format for Nzoia, Malaba and Nyando basins The flood maps generated comes in geo-tiff format which can be read in any GIS environment
